## 二、前端技术栈

### 2.1 核心框架
- **Vue 3.4+**：Composition API，更好的TypeScript支持
- **TypeScript 5.0+**：类型安全
- **Vite 5.0+**：极速开发体验

### 2.2 UI组件库
- **Element Plus 2.5+**：企业级组件库
- **定制主题**：爱马仕橙 (#FF6600)
- **图标**：@element-plus/icons-vue

### 2.3 状态管理
- **Pinia**：Vue 3官方推荐
- **pinia-plugin-persistedstate**：状态持久化

### 2.4 路由管理
- **Vue Router 4**：动态路由、权限控制

### 2.5 HTTP请求
- **Axios**：统一封装、拦截器

### 2.6 文件上传
- **vue-upload-component**：大文件分片上传

### 2.7 富文本编辑
- **Quill / TinyMCE**：报告编辑

### 2.8 图表可视化
- **ECharts 5**：数据统计图表

### 2.9 工具库
- **Day.js**：日期处理
- **Lodash-es**：工具函数
- **XLSX**：Excel导入导出
- **jsPDF**：PDF生成

## 三、后端技术栈

### 3.1 核心框架
- **Python 3.11+**
- **FastAPI 0.109+**：高性能异步框架
- **Uvicorn**：ASGI服务器
- **Gunicorn**：进程管理

- **SQLAlchemy 2.0**：ORM框架
- **Alembic**：数据库迁移

- **Celery 5.3+**：异步任务队列
- **Flower**：Celery监控

### 3.4 文件存储
- **MinIO**：私有化对象存储
- **阿里云OSS**：云存储备选

### 3.5 AI相关
- **PaddleOCR**：开源OCR引擎
- **百度OCR API**：商用OCR备选
- **OpenAI API**：GPT-4模型
- **通义千问API**：国产大模型备选
- **LangChain**：LLM应用框架
- **Faiss**：向量数据库

### 3.6 文档处理
- **PyPDF2 / pdfplumber**：PDF解析
- **openpyxl**：Excel处理
- **python-docx**：Word生成
- **Pillow**：图像处理

### 3.7 认证授权
- **JWT**：Token认证
- **bcrypt**：密码加密

### 3.8 日志监控
- **Loguru**：日志记录
- **Sentry**：错误追踪
- **Prometheus + Grafana**：性能监控

## 四、数据库选型

### 4.1 PostgreSQL（主数据库）
**选择理由**：
- 强大的JSON支持
- 全文检索能力
- 事务ACID保证

**存储内容**：
- 用户、权限、组织架构
- 项目、客户信息
- 租赁物清单、发票记录
- 尽调报告、合同数据
- 操作日志、审计日志

### 4.2 Redis（缓存与队列）
**使用场景**：
- 用户会话Session
- 热点数据缓存
- 分布式锁
- 消息队列（Celery Broker）

### 4.3 MinIO（对象存储）
**存储内容**：
- 上传的原始文件
- AI处理后的文件
- 生成的报告文档
- 用户头像等静态资源

### 4.4 Elasticsearch（可选）
**使用场景**：
- 全文检索
- 日志分析

### 4.5 Faiss（向量数据库）
**使用场景**：
- 文档向量化存储
- 语义相似度检索
- 智能问答知识库

## 五、部署架构

### 5.1 开发环境
```
Docker Compose 一键启动
- Web前端：Vite Dev Server (5173端口)
- API后端：Uvicorn (8000端口)
- PostgreSQL：5432端口
- Redis：6379端口
- MinIO：9000端口
```

### 5.2 生产环境（私有化部署）
```
┌─────────────────────────────────────┐
│         Nginx (80/443)              │
│  SSL证书、静态资源、反向代理          │
└──────────┬──────────────────────────┘
           │
    ┌──────┴──────┐
    │             │
┌───▼────┐   ┌───▼────┐
│ Vue前端 │   │FastAPI │
│ (静态)  │   │ 后端   │
└────────┘   └───┬────┘
                 │
    ┌────────────┼────────────┐
    │            │            │
┌───▼───┐  ┌────▼───┐  ┌─────▼────┐
│Postgre│  │ Redis  │  │  MinIO   │

```

### 5.3 SaaS云服务部署
```
阿里云 / 腾讯云
- ECS：应用服务器（2核4G起）
- RDS：PostgreSQL托管服务
- Redis：Redis托管服务
- OSS：对象存储
- SLB：负载均衡
- CDN：静态资源加速
```
