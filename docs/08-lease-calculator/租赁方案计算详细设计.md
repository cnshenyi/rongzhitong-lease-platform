# 融智通 - 模块5：租赁方案计算 详细设计

## 一、模块概述

### 1.1 业务背景
融资租赁项目中，业务员需要根据客户需求，快速生成融资方案报价：
- 客户：融资多少钱？每期还多少？利率是多少？
- 业务员：手动用Excel计算，容易出错，效率低
- 领导审批：需要标准化的方案文件

1. 多种还款方式计算（等额租金、等额本金、不规则还款等）

3. 生成客户报价单（PDF）
4. 方案对比功能
5. 关联IRR/收益率自动计算

## 三、计算模型

### 3.1 还款方式

#### 3.1.1 等额租金（PMT）
```
每期租金固定，包含本金和利息

公式：
PMT = PV × r × (1+r)^n / ((1+r)^n - 1)

其中：
  PV = 融资本金
  r  = 每期利率（年利率/期数）
  n  = 总期数

示例：
  融资金额：500万元
  年利率：8%
  期限：36个月（月付）
  
  月利率 r = 8% / 12 = 0.6667%
  PMT = 5,000,000 × 0.006667 × (1.006667)^36 
        / ((1.006667)^36 - 1)
      = 156,688.58 元/月
```

#### 3.1.2 等额本金
```
每期本金固定，利息逐期递减

公式：
每期本金 = PV / n
第k期利息 = (PV - PV/n × (k-1)) × r
第k期租金 = 每期本金 + 第k期利息

特点：
- 前期租金高，后期租金低
- 总利息比等额租金少
```

#### 3.1.3 先息后本
```
前期只付利息，最后一期归还本金

公式：
前n-1期：每期租金 = PV × r（仅利息）
第n期：租金 = PV × r + PV（利息+全部本金）

适用场景：
- 项目前期资金紧张
- 设备投入运营后才有收入
```

#### 3.1.4 不规则还款
```
自定义每期还款金额

特点：
- 灵活匹配客户现金流
- 支持设置宽限期（前几期少还或不还）
- 支持季付/半年付/年付
- 支持混合付款方式
```

### 3.2 IRR计算

```python
# 内部收益率计算
def calculate_irr(cash_flows):
    """
    cash_flows: 现金流列表
    [初始投资(负), 第1期收入, 第2期收入, ...]
    
    例如：
    cash_flows = [-5000000, 156689, 156689, ..., 156689]
                  初始放款    第1期    第2期        第36期
    """
    # 使用numpy-financial计算
    import numpy_financial as npf
    
    irr = npf.irr(cash_flows)
    annual_irr = irr * 12  # 月IRR转年IRR
    
    return annual_irr

# 考虑手续费的综合收益率
def calculate_comprehensive_irr(cash_flows, fees):
    """
    综合收益率 = 考虑各类费用后的实际收益率
    
    费用类型：
    - 顾问费（前端收取）
    - 手续费（前端收取）
    - 保证金（租赁结束退还）
    - 管理费（每期收取）
    """
    adjusted_flows = cash_flows.copy()
    
    # 前端费用加到第0期
    adjusted_flows[0] = adjusted_flows[0] + fees["advisory_fee"] + fees["service_fee"]
    adjusted_flows[0] = adjusted_flows[0] + fees["deposit"]  # 保证金
    
    # 保证金在最后一期退还
    adjusted_flows[-1] = adjusted_flows[-1] - fees["deposit"]
    
    # 每期管理费
    for i in range(1, len(adjusted_flows)):
        adjusted_flows[i] += fees["management_fee_per_period"]
    
    irr = npf.irr(adjusted_flows)
    return irr * 12
```

## 六、提前还款测算

```python
def calculate_prepayment(schedule, prepay_period, prepay_type):
    """
    提前还款测算
    
    prepay_period: 提前还款的期数（如第12期提前还款）
    prepay_type: 
        - "full": 全额提前还款
        - "partial": 部分提前还款
    """
    remaining_principal = schedule[prepay_period]["remaining_principal"]
    
    if prepay_type == "full":
        # 全额提前还款
        # 需支付：剩余本金 + 违约金（通常为剩余本金的1-3%）
        penalty_rate = 0.02  # 2%违约金
        penalty = remaining_principal * penalty_rate
        total_prepay = remaining_principal + penalty
        
        # 节省利息
        total_remaining_interest = sum(
            s["interest"] for s in schedule[prepay_period:]
        )
        saved_interest = total_remaining_interest
        
        return {
            "remaining_principal": remaining_principal,
            "penalty": penalty,
            "total_prepayment": total_prepay,
            "saved_interest": saved_interest
        }
```

## 七、异常处理

| 异常类型 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 参数越界 | 利率>36%或金额为负 | 表单校验提示 |
| IRR无解 | 现金流无法收敛 | 提示参数不合理 |
| 精度问题 | 小数点精度丢失 | 使用Decimal库计算 |
| 导出失败 | PDF生成错误 | 重试，降级为Excel |
