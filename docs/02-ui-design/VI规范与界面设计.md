# 融智通 - VI规范与界面设计

## 一、品牌VI规范

### 1.1 品牌色彩

#### 主色调：爱马仕橙
```
主色：#FF6600 (Hermès Orange)
RGB: (255, 102, 0)
CMYK: (0, 60, 100, 0)

使用场景：
- 主按钮
- 重要图标
- 强调文字
- 进度条
- 选中状态
```

#### 辅助色
```
深橙色：#E65500 (hover状态)
浅橙色：#FF8533 (背景、边框)
极浅橙：#FFF3E6 (背景色块)
```

#### 中性色
```
深灰：#333333 (主文字)
中灰：#666666 (次要文字)
浅灰：#999999 (辅助文字)
边框灰：#DCDFE6
背景灰：#F5F7FA
白色：#FFFFFF
```

#### 功能色
```
成功：#67C23A (绿色)
警告：#E6A23C (黄色)
危险：#F56C6C (红色)
信息：#409EFF (蓝色)
```

### 1.2 字体规范

#### 中文字体
```
优先级：
1. PingFang SC (苹果系统)
2. Microsoft YaHei (Windows)
3. sans-serif (兜底)
```

#### 英文/数字字体
```
优先级：
1. Helvetica Neue
2. Arial
3. sans-serif
```

#### 字号规范
```
特大标题：28px / 32px
大标题：24px / 28px
中标题：20px / 24px
小标题：18px / 22px
正文：14px / 20px
辅助文字：12px / 18px
```

### 1.3 圆角规范
```
大圆角：8px (卡片、弹窗)
中圆角：4px (按钮、输入框)
小圆角：2px (标签、徽章)
```

### 1.4 阴影规范
```
轻阴影：0 2px 4px rgba(0, 0, 0, 0.08)
中阴影：0 4px 12px rgba(0, 0, 0, 0.12)
重阴影：0 8px 24px rgba(0, 0, 0, 0.16)
```

### 1.5 间距规范
```
超小：4px
小：8px
中：16px
大：24px
超大：32px
```

## 五、响应式设计

### 5.1 断点定义
```
超小屏：< 768px（手机）
小屏：768px - 1024px（Pad竖屏）
中屏：1024px - 1440px（Pad横屏、小笔记本）
大屏：1440px - 1920px（常规显示器）
超大屏：> 1920px（2K、4K显示器）
```

### 5.2 适配策略

#### 手机端（< 768px）
- 左侧菜单改为抽屉式
- 表格改为卡片列表
- 隐藏部分次要信息
- 按钮全宽显示

#### Pad端（768px - 1024px）
- 左侧菜单可折叠
- 表格正常显示
- 适当调整间距

#### PC端（> 1024px）
- 完整功能展示
- 最佳体验
