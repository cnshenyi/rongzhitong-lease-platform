# 融智通 - 模块3：尽调报告撰写 详细设计

## 一、模块概述

### 1.1 业务背景
融资租赁项目立项前，需要对承租人（客户）进行尽职调查（Due Diligence），
撰写尽调报告是风控的核心环节。

传统尽调报告撰写：
- 需要收集大量资料（工商信息、财务报表、征信报告、行业资料等）
- 人工整理分析，撰写报告
- 耗时3-7天，报告质量依赖分析师经验

### 1.2 核心功能
通过AI自动化完成：
1. 自动收集企业公开信息（工商、司法、舆情等）
2. 分析财务数据，生成财务分析
3. 行业研究和竞争分析
4. LLM自动撰写结构化尽调报告
5. 支持人工审核和修改

### 1.3 价值量化
| 指标 | 现状 | 使用后 | 提升 |
|------|------|--------|------|
| 报告撰写时间 | 3-7天 | 2-4小时 | 20倍 |
| 信息收集完整度 | 70% | 95%+ | +25% |
| 报告标准化程度 | 低（看人） | 高（模板化） | 显著提升 |
| 人力成本 | 1人×5天 | 0.5人×0.5天 | -90% |

#### 步骤3：AI分析处理

**3.1 财务分析**
```python
# 自动计算财务指标
indicators = {
    "偿债能力": {
        "资产负债率": (总负债 / 总资产) * 100,
        "流动比率": 流动资产 / 流动负债,
        "速动比率": (流动资产 - 存货) / 流动负债
    },
    "盈利能力": {
        "净利润率": (净利润 / 营业收入) * 100,
        "ROE": (净利润 / 净资产) * 100,
        "ROA": (净利润 / 总资产) * 100
    },
    "营运能力": {
        "应收账款周转率": 营业收入 / 平均应收账款,
        "存货周转率": 营业成本 / 平均存货
    },
    "成长能力": {
        "营收增长率": (本年营收 - 上年营收) / 上年营收 * 100,
        "利润增长率": (本年利润 - 上年利润) / 上年利润 * 100
    }
}

# LLM生成财务分析文本
prompt = f"""
根据以下财务指标，撰写财务分析：
{indicators}

要求：
1. 分析各项指标的含义和水平
2. 与行业平均值对比
3. 指出财务优势和风险点
4. 给出专业评价
"""
```

**3.2 风险评估**
```python
# 风险评分模型
risk_score = {
    "信用风险": {

        "诉讼记录": 20分,  # 涉诉情况

    "财务风险": {
        "资产负债率": 25分,  # >70%高风险
        "盈利能力": 20分,    # 连续亏损高风险
        "现金流": 15分       # 现金流紧张高风险
    },
    "经营风险": {
        "行业景气度": 15分,
        "市场地位": 10分,
        "客户集中度": 10分
    }
}

# 综合评分：0-100分
# 90-100：优秀，低风险
# 70-89：良好，中低风险
# 50-69：一般，中等风险
# <50：较差，高风险
```
