## 一、行业背景

### 1.1 融资租赁行业概况
- 全国融资租赁公司数量：约10,000+家（含金融租赁、商业租赁）
- 行业总资产规模：约7万亿元
- 从业人员：约30万人
- 数字化程度：整体偏低，大部分依赖Excel和纸质文档

### 1.2 核心业务流程
```
客户申请 → 项目立项 → 尽职调查 → 风险审批 
→ 合同签署 → 放款执行 → 租后管理 → 到期处置
```

### 1.3 行业痛点

#### 文档处理效率低下
- 租赁物清单格式混乱（Excel、PDF、扫描件、照片）
- 业务员手动录入，平均每项目耗时4-8小时
- 几百上千条设备明细逐一核对

#### 风险控制手段落后
- 发票真伪靠人工逐张查验
- 跨项目发票重复使用难以发现
- 欺诈案件年损失可达数百万

#### 尽调报告质量参差
- 信息采集耗时2-3天
- 报告质量依赖个人经验
- 格式不统一，合规性难保证

#### 人力成本高昂
- 一个业务员同时处理5-8个项目
- 70%时间花在重复性文档工作上
- 专业人才招聘困难、流动性高

## 二、产品定位

### 2.1 我们是什么
融智通是一款**垂直于融资租赁行业的AI SaaS工具平台**。

不是一个大而全的ERP系统，而是**聚焦高频痛点场景**的效率工具：
- 租赁物清单识别
- 发票查重验真
- 尽调报告撰写
- 合同智能审核
- 租赁方案计算
- 智能问答助手

### 2.2 我们不是什么
- 不是融资租赁ERP/核心系统（不替代金蝶、用友等）
- 不是CRM客户管理系统
- 不是财务系统
- 不做业务审批流程（但可对接）

### 2.3 与现有系统的关系

## 三、目标用户画像

### 3.1 主要目标：中小型融资租赁公司
- **规模**：10-100人
- **年业务量**：5-50亿
- **特征**：人手不足，效率要求高，IT预算有限
- **决策人**：总经理/业务总监
- **使用人**：业务员、风控人员
- **预算**：5,000-30,000元/年

### 3.2 次要目标：银行系金融租赁公司
- **规模**：大型机构的业务条线/部门
- **特征**：流程标准化要求高，风控严格
- **决策人**：部门总经理/IT负责人
- **使用人**：业务团队
- **预算**：30,000-100,000元/年（或私有化部署15-50万）

### 3.3 扩展目标
- 设备租赁公司
- 商业保理公司
- 供应链金融公司
